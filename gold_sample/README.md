# Gold Sample Coding Files

Four CSV recording sheets — one per pipeline pass. Import all four into a single spreadsheet workbook (Google Sheets or Excel) as separate tabs.

| File | Unit | When to use |
|---|---|---|
| pass1_theories.csv | One row per theory per article | While reading introduction |
| pass2_hypotheses.csv | One row per hypothesis per article | While reading introduction (end) |
| pass3_discussion.csv | One row per article | While reading discussion |
| pass4_methods.csv | One row per article | While reading methods |

**Before coding:** Read VALIDATION_MANUAL.md in full.  
**Valid options per field:** See VALIDATION_MANUAL.md Section 4–7.  
**IRR target:** κ ≥ 0.61 on all primary fields.  
**Do not discuss classifications with the other coder until both have finished all 30 articles.**
